# Docfork

Up-to-date documentation for AI coding agents. 10,000+ libraries formatted for LLMs.

### Repos

- [Docfork MCP](https://github.com/docfork/docfork) · MCP Server (Cursor, Claude Code, OpenCode, Windsurf)

### Connect

```json
{
  "mcpServers": {
    "docfork": { "url": "https://mcp.docfork.com/mcp" }
  }
}
```

Or run locally:

```bash
npx -y docfork
```

Then ask your agent:

> *"Create a Next.js App Router app. use docfork"*
